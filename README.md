# 📡 Программная модель передачи данных  
## (MECH, MANET, FONBETA, ASTM F38)

Программа моделирует физический уровень связи для четырёх стандартов.  
Реализованы модуляции **BPSK**, **QPSK**, **16QAM**, канал с **AWGN** шумом, расчёт **BER**, визуализация созвездий и передача изображений.

---

## ⚙️ Возможности

- Выбор стандарта и типа модуляции
- Построение графика **BER(SNR)** для одного или всех стандартов
- Визуализация **созвездия сигнала** (передатчик / приёмник)
- Передача **изображения** через шумный канал

---

## 🚀 Быстрый старт

Установи библиотеки:

```bash
pip install numpy matplotlib pillow
```

Запусти программу:
python transmission_model.py
